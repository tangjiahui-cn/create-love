# React Components Library.

## How To Run ?
run a local server.
```shell
# install dependencies
pnpm install

# start local demo.
pnpm dev

# start local docs site.
pnpm docs
```
build library.
```shell
# build ui-library.
pnpm build

# build docs-site.
pnpm build:docs
```
publish library.
```shell
# publish ui-library.
pnpm publish
```
