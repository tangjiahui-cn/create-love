# easy-cli
It's easy to create a empty project.

## How To Use?
show all projects.
```shell
easy ls

# easy ls
# - components-ts-vue
# - components-ts-react
# - lib-ts
# - lib-js
```

## Next Plan
```shell
# create a project step by step.
easy init

# create a project from template to target directory.
easy create <template-name> [target]
```

## Run Development
```shell
# install dependencies.
pnpm install

# link to global env.
npm link

# run dev-bin.
easyl ls
```
